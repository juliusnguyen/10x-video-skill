Create HTML slides based on the script.

Usage: `/slides <N>`

Steps:
1. Use the v5.1 design system (see `references/design-system.md`).
2. Map slides to script sections.
3. Use SVG icons only (no emojis).
4. Use JS-driven animations.
5. **Brand Ending:** Ensure the final CTA slide uses the "Brand Name" from `GEMINI.md`.
6. Save as `video{N}_{name}.html`.
